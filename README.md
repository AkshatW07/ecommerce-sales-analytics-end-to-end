
📊E-Commerce Product Analytics Dashboard



Project Overview
This project analyzes e-commerce product data to uncover insights related to pricing, discounts, and product performance. The goal is to help businesses make data-driven decisions to improve revenue and customer satisfaction.

 Problem Statement
How can an e-commerce platform improve revenue and product performance using data?



🛠 Tools & Technologies

Python (Pandas, NumPy)
SQL (MySQL)
Power BI
Excel




🔄 Project Workflow

Data Cleaning & Preprocessing (Python)
Database Creation & ETL (SQL)
Exploratory Data Analysis
Dashboard Development (Power BI)





📈 Key Insights

High-priced products do not always have better ratings
Heavy discounts are concentrated in specific categories
A small number of brands dominate product listings
Most products fall within the mid-price range



📊 Dashboard Features

KPI cards for key metrics (Price, Rating, Discount, Products)
Category-wise pricing analysis
Discount strategy visualization
Top-performing brands analysis
Price vs Rating relationship (scatter plot)
Interactive filters (Category & Brand)


💡 Business Impact

This dashboard enables businesses to:

Optimize pricing strategies
Improve discount effectiveness
Identify high-performing products and brands
Enhance customer satisfaction
